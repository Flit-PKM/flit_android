# Welcome to Flit

Flit is your fast capture workspace for ideas, notes, and voice thoughts.
You can type notes, record voice notes, connect related notes, and sync with Flit - Core.

## Getting started

### Create a note

- Type in the input bar at the bottom of the Home screen and tap send.
- Or hold the record button to create a voice note (only when a transcription model is selected).

### Voice Transcription

Choose the model that fits your device:

- None: text only, no voice transcription
- Light: fast and smaller
- Medium: balanced
- Performance: most accurate, largest download

You can change this later in Settings.

### Note actions on Home

Each note card has action buttons:

- Append: start a follow-up note related to this note
- Edit: open the note editor
- Delete: delete the note

### Note view

Tap a note card to open note view. From there you can:

- read content
- open related notes
- open note categories
- append, edit, or delete the note

### Note Edit view

Tap the Edit button to edit a note, here you can:

- edit note content with markdown formatting
- add and remove note categories
- create or delete relationships between different notes

### Settings

In Settings you can configure:

- theme mode
- note preview details
- transcription model
- export and import
- connection and sync with Flit - Core

### Connect to Flit - Core

To enable sync:

1. Tap "[Log in on Flit - Core](https://core.flit-pkm.com)"
2. Request a connection code
3. Enter the code in the app
4. Use "Sync with Flit - Core" in Settings

Syncing with Flit - Core allows:

- Full note backup
- Multi device sync
- Voice note formatting (Coming Soon!)
- AI enhanced note categorization and relationship analysis (Coming Soon!)
